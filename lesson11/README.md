# Day11 · 函数进阶

## 关键知识点
- 默认参数:`def f(x, n=2)`,有默认值的参数必须靠后
- `*args`:收集多余位置参数为元组,实现可变个数参数
- `**kwargs`:收集多余关键字参数为字典- 混合参数顺序:位置 → `*args` → `**kwargs`- `lambda` 匿名函数 + `sort(key=...)` 排序
- 组包 / 解包:函数内部的元组/字典打包
