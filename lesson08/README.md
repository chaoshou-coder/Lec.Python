# Day08 · 字符串提升

## 关键知识点
- 字符串查找：`find()`、`index()`、`rfind()`
- 字符串判断：`endswith()`、`startswith()`
- 字符串替换：`replace()`
- 字符串拆分与拼接：`split()`、`join()`
- 字符串格式化：`title()`、`strip()`、`lstrip()`、`rstrip()`
- 循环 + 字符串综合处理
