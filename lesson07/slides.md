## 1. 第一讲(15 分钟) —— 为什么需要函数 + 定义与调用

### 知识点 1.1 从"重复代码"到"打包成工具"

写代码和做饭一样 —— 如果每次切菜都临时找刀,不如把刀放在工具箱里,用的时候直接拿。函数就是把一段逻辑"命名 + 打包",以后用名字就能调用。

**没有函数的写法**(重复):
```python
# 给小明打印问候语
print("==========")
print("你好, 小明!")
print("欢迎学习 Python!")
print("==========")

# 给小红打印问候语 —— 复制粘贴,改个名字
print("==========")
print("你好, 小红!")
print("欢迎学习 Python!")
print("==========")
```

**有函数的写法**(一次定义,多次调用):
```python
def 打招呼(名字):
    """打印一行问候语"""
    print("==========")
    print(f"你好, {名字}!")
    print("欢迎学习 Python!")
    print("==========")

# 调用两次,传不同名字
打招呼("小明")
打招呼("小红")
```

> 口诀:**定义一次,调用无数次**。函数 = 给一段代码起名字。

### 知识点 1.2 函数定义的语法规则

`def` 是 define(定义)的缩写。完整结构:

```python
def 函数名(参数1, 参数2, ...):
    """文档字符串(可选,说明这个函数干啥)"""
    # 函数体(必须缩进 4 格)
    return 返回值  # 可选,没有就返回 None
```

四个关键细节:
1. **`def` 后面必须有冒号 `:`** —— 和 `if`、`for` 一样,冒号表示"下面开始是新块"。
2. **函数体必须缩进** —— 缩进 4 格,和 `if`/`for` 块规则相同。
3. **定义不等于执行** —— 只 `def` 不调用,函数体里的代码**一行都不跑**。
4. **调用必须加括号** —— `打招呼` 只是函数名(变量),`打招呼()` 才是"执行"。

```python
def 说你好():
    print("你好!")

说你好   # 啥也没发生,只是拿到函数对象
说你好()  # 你好!  ← 这才是调用
```

> 🔴 教学红线(定义 vs 调用): 学员常以为 `def` 完就执行了。演示: 只写 `def 测试(): print("跑了")` 不调用,终端无任何输出 —— 强调"定义只是把工具做好,调用才是拿出来用"。

## 3. 第二讲(15 分钟) —— 四种形式 + return 的两个作用

### 知识点 3.1 函数的四种形式

根据"有没有参数"和"有没有返回值"组合,函数有四种形式。本节重点掌握**有参有返回**。

```python
# 形式 1: 无参无返回(做"动作",比如打印)
def 打印分割线():
    print("-" * 20)

# 形式 2: 有参无返回(做"动作",但动作内容由参数决定)
def 打印分割线(长度):
    print("-" * 长度)

# 形式 3: 无参有返回(做"计算",从外部拿数据)
def 获取当前年份():
    return 2026

# 形式 4: 有参有返回(最常用 —— 输入 → 处理 → 输出)
def 计算总价(单价, 数量):
    return 单价 * 数量
```

> 教学顺序建议: 先讲形式 4(最实用),再回头告诉学员"其他三种是它的简化版"。避免四种一起讲导致信息过载。

### 知识点 3.2 `return` 的两个作用

`return` 在函数里做两件事 —— **返回值** + **提前退出**。

**作用 1: 返回值给调用方**
```python
def 计算总价(单价, 数量):
    return 单价 * 数量

结果 = 计算总价(5.5, 3)  # 结果 = 16.5
print(结果)              # 16.5
```

**作用 2: 提前退出函数**
函数执行到 `return` 就**立刻结束**,后面的代码不执行。

```python
def 检查年龄(年龄):
    if 年龄 < 0:
        print("年龄不能为负数!")
        return              # ← 提前退出,下面两行不执行
    print(f"你的年龄是 {年龄}")
    print("检查完毕!")

检查年龄(-5)  # 只打印"年龄不能为负数!"
检查年龄(18)  # 打印"你的年龄是 18"和"检查完毕!"
```

> 口诀:**return 是函数的"出口"** —— 可以带值出去(返回值),也可以空着手出去(提前退出)。

### 知识点 3.3 形参 vs 实参:占位符 vs 传入的值

```python
def 计算总价(单价, 数量):  # ← 单价、数量 是"形参"(形式参数)
    return 单价 * 数量

计算总价(5.5, 3)           # ← 5.5、3 是"实参"(实际参数)
```

- **形参** = 函数定义时的占位符,相当于"坑位"。
- **实参** = 函数调用时填进去的实际值,相当于"填坑"。
- 调用时实参按**位置**一一对应传给形参,所以顺序很重要。

```python
计算总价(5.5, 3)   # 单价=5.5, 数量=3 → 16.5
计算总价(3, 5.5)   # 单价=3, 数量=5.5 → 16.5(数值一样,但语义错了!)
```

> 🔴 教学红线(返回函数 vs 返回值 —— 预告): 本节不展开,但可提一句 —— `return 函数名` 是返回函数本身(一个工具),`return 函数名()` 是调用函数并返回它的结果(一个值)。Day08 会详细讲。

## 5. 第三讲(15 分钟) —— 参数 vs 实参深入 + 函数与旧知识结合

### 知识点 5.1 函数里可以用所有 Day01-Day06 的工具

函数不是新知识,而是**旧知识的组合容器**。函数体里可以写:
- `if` / `elif` / `else` —— 分支
- `for` / `while` —— 循环
- 字符串方法 —— `find()` / `replace()` / 切片
- 列表操作 —— `append()` / `pop()` / 遍历
- f-string —— 格式化输出

```python
def 打印购物车(购物车):
    """接收一个购物车列表,打印每条商品"""
    if len(购物车) == 0:
        print("购物车为空!")
        return
    for i in range(len(购物车)):
        商品 = 购物车[i]
        小计 = 商品[1] * 商品[2]
        print(f"{i + 1}. {商品[0]} {商品[1]:.2f} x {商品[2]} = {小计:.2f}")

购物车 = [["苹果", 5.50, 3], ["牛奶", 12.80, 2]]
打印购物车(购物车)
```

> 教学提示: 这个例子把 Day04 列表 + Day05 循环 + Day06 分支 + Day03 f-string 全部"装"进了一个函数 —— 这就是函数的威力。

### 知识点 5.2 多个参数 + 多返回值(元组解包)

函数可以接收多个参数,也可以"返回多个值"(实际是返回一个元组)。

```python
def 计算成绩(语文, 数学, 英语):
    """返回总分和平均分"""
    总分 = 语文 + 数学 + 英语
    平均分 = 总分 / 3
    return 总分, 平均分   # ← 实际返回 (总分, 平均分) 元组

结果 = 计算成绩(90, 85, 95)
print(结果)          # (270, 90.0)

总分, 平均分 = 计算成绩(90, 85, 95)  # ← 元组解包
print(f"总分 {总分}, 平均分 {平均分:.1f}")
```

### 知识点 5.3 函数调用的执行流程(内存图)

用一张图讲清"调用时发生了什么":

```python
def 加倍(x):
    return x * 2

数值 = 10
结果 = 加倍(数值)
print(结果)  # 20
```

执行流程:
1. `数值 = 10` —— 把 10 存到变量 `数值`。
2. `加倍(数值)` —— 调用函数,把 `数值` 的值(10)复制一份给形参 `x`。
3. `return x * 2` —— 计算 10 * 2 = 20,返回给调用方。
4. `结果 = ...` —— 把返回值 20 存到变量 `结果`。

> 关键:**形参 `x` 只是实参 `数值` 的"复印件"**,函数里改 `x` 不会影响 `数值`(Day08 会讲列表作为参数时情况不同)。

## 易错点

1. **定义不等于执行**: 只 `def` 不调用,函数体里的代码一行都不跑;调用必须加括号 `()`。
2. **缩进错误**: 函数体必须缩进 4 格,否则报 `IndentationError`。
3. **`return` 是出口**: 执行到 `return` 函数立刻结束,后面代码不执行;`return` 不带值时返回 `None`。
4. **形参 vs 实参**: 形参是定义时的占位符,实参是调用时填进去的值;按位置一一对应,顺序很重要。
5. **`return` vs `print`**: `return` 是把值"交给调用方"使用,`print` 是把值"给人看"输出到终端;函数里要返回结果用 `return`,不要 `print`。
