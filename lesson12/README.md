# Day12 · 迭代 / 内置 / 递归

## 关键知识点
- `lambda` 匿名函数:短小的单行函数
- `filter()` + lambda:按条件过滤元素
- `map()` + lambda:对每个元素做转换
- `sorted()` + `reverse` / `key=lambda` 排序
- `all()` 全 True / `any()` 任一 True
- `global` / `globals()`:访问全局变量
- 递归:基线条件 + 递归调用 (累加 / 阶乘 / 斐波那契)
