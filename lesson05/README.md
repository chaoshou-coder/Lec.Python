# Day05 · 循环入门

## 关键知识点
- `while` 循环与循环控制（`break`、`continue`）
- `for` 循环 + `range()`（开始、结束、步长）
- 转义符：`\n`、`\t`
- `end=` 参数控制输出不换行
- 嵌套循环（乘法表、图形打印）
- 累加、累乘与计数技巧
