# Day03 · 字符串基础

## 关键知识点
- 字符串索引与切片：`s[i]`、`s[start:end]`
- `len()` 获取字符串长度
- 字符串查找：`find()`、`rfind()`
- 字符串判断：`isdigit()`、`isalpha()`、`isalnum()`
- 字符串替换：`replace()`
- 格式化输出：`%`、`format`、`f-string`
