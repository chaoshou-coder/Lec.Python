## 1. 第一讲(15 分钟) —— 字符串索引、切片与长度

### 知识点 1.1 字符串索引:给每个字符编上门牌号

字符串和现实中的"排队"一样,每个字符都有一个从 0 开始的编号(索引)。

```python
name = "孙悟空"
print(name[0])   # 孙  ← 第 1 个字符,索引是 0
print(name[1])   # 悟
print(name[2])   # 空
print(name[-1])  # 空  ← 倒数第 1 个,负数索引从右往左数
print(name[-2])  # 悟
```

> 口诀:**从零开始,负数倒着数**。
>
> 🔴 教学红线(零索引困惑): 学员常把"第几个"和"索引"搞混。用"门牌号 vs 距离"类比 —— 第 1 家住门牌号 0,第 2 家住门牌号 1,就像"距离起点 0 步"vs"距离起点 1 步"。建议画内存图:画一排格子,顶上标索引,底下标字符。

### 知识点 1.2 切片:一次切出一段

`s[start:end]` 从 `start` 切到 `end` **之前**(左闭右开),就像切蛋糕 —— 刀口落在 `end`,但 `end` 那块不拿。

```python
s = "hello world"
print(s[0:5])    # hello
print(s[6:11])   # world
print(s[:5])     # hello  ← 省略 start 默认从 0 开始
print(s[6:])     # world  ← 省略 end 默认切到末尾
print(s[:])      # hello world  ← 全切,相当于复制
```

### 知识点 1.3 `len()` 数一数字符串有多长

```python
print(len("hello"))       # 5
print(len("你好"))         # 2  ← 一个汉字算一个字符,不是字节
print(len(""))             # 0  ← 空字符串长度为 0
```

> 易混:`len()` 数的是**字符个数**,不是字节数。初学者常把"占多少字节"和"有几个字"搞混,本节只讲字符个数。

## 3. 第二讲(15 分钟) —— 字符串查找与判断

### 知识点 3.1 `find()` 与 `rfind()`:正着找 vs 倒着找

`find(sub)` 返回子串 `sub` **第一次**出现的索引;`rfind(sub)` 返回**最后一次**出现的索引。找不到都返回 `-1`。

```python
s = "hello hello hello"
print(s.find("hello"))   # 0   ← 第一次
print(s.rfind("hello"))  # 12  ← 最后一次
print(s.find("python"))  # -1  ← 找不到
```

> 为什么需要两个? 例: 文件路径 `/tmp/data/a.txt` 要取最后一个 `/` 之后的内容,只能倒着找 —— `rfind("/")`。

### 知识点 3.2 字符串判断方法:是不是数字?是不是字母?

这三个方法都返回 `True` / `False`,常和 `if` 搭配做输入校验。

```python
print("12345".isdigit())    # True  ← 全是数字
print("12.3".isdigit())     # False ← 小数点不是数字
print("abc".isalpha())      # True  ← 全是字母
print("abc123".isalpha())   # False
print("abc123".isalnum())   # True  ← 字母或数字(只要不是符号就行)
print("abc-123".isalnum())  # False ← 横杠是符号
```

> 记忆口诀:**digit = 数字键,isalpha = 字母表,alnum = 字母+数字**。

## 5. 第三讲(15 分钟) —— 字符串替换与格式化输出

### 知识点 5.1 `replace()`:批量替换

`str.replace(old, new)` 把字符串中**所有** `old` 子串替换成 `new`,返回新字符串(原串不变 —— 字符串是不可变的)。

```python
text = "xxx 是个好网站,xxx 很好用"
clean = text.replace("xxx", "***")
print(clean)  # *** 是个好网站,*** 很好用

# 替换次数可以限制(可选参数,了解即可)
print("aaa aaa aaa".replace("aaa", "b", 2))  # b b aaa
```

> 注意:`replace()` 是**全量替换**,不是只替第一个。想替第一个要加第三个参数 `count=1`。
>
> 🔴 教学红线(字符串不可变性): 学员常以为 `replace()` 修改了原串。演示 `s = "abc"; s.replace("a", "x"); print(s)` 仍输出 `abc` —— 必须用变量接住返回值。进一步演示 `s[0] = "x"` 报错 `TypeError`,说明字符串是"只读"的。

### 知识点 5.2 格式化输出:三种"填空"方式

把变量"塞"进字符串,本节学三种,重点掌握 **f-string**(最常用)。

```python
name = "小明"
age = 18

# 方式 1:% 格式化(古老,C 语言风格,了解即可)
print("我是 %s,今年 %d 岁" % (name, age))

# 方式 2:str.format()(Python 2.7+,熟悉)
print("我是 {},今年 {} 岁".format(name, age))

# 方式 3:f-string(Python 3.6+,最推荐,本节重点)
print(f"我是 {name},今年 {age} 岁")
```

> 教学顺序建议: 先讲 f-string(直观,大括号里直接写变量),再回头告诉学员"你偶尔会在老代码里看到 `%` 和 `format`,知道是同样的东西就行"。避免三种一起讲导致混淆。
>
> 🔴 教学红线(字符串拼接性能): 大量拼接时 `+` 效率低(每次都创建新对象),推荐用 f-string 或 `join()`。本节只需点出"f-string 是最佳实践",不必展开性能 benchmark。

## 易错点

1. **索引差一**: 第 n 个字符的索引是 `n-1`;切片 `s[a:b]` 取到 `b` 之前,不含 `b`。
2. **`find()` 找不到返回 `-1`**,不是 `None` 也不是 `0`;判断"找不到"要写 `== -1`。
3. **字符串不可变**: `replace()` 返回新字符串,原串不变,必须用变量接住返回值。
4. **`isdigit()` 只认纯数字**: 负数 `"-1"`、小数 `"1.5"`、全角数字 `"１２"` 都返回 `False`。
5. **f-string 大括号里是表达式**,不能写语句(不能在大括号里赋值)。
