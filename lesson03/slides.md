## 1. 第一讲(15 分钟) —— 比较运算符与 `if` 基础

### 知识点 1.1 比较运算符:两个值比大小

比较运算符两边的表达式计算后返回 **`True` / `False`**(布尔值)。

```python
print(3 > 1)    # True
print(3 < 1)    # False
print(3 == 3)   # True   ← 注意是双等号!(单等号是赋值)
print(3 != 4)   # True   ← 不等
print(3 >= 3)   # True
print(3 <= 2)   # False
```

> 🔴 教学红线(`=` vs `==`): 学员最经典错误是在 `if` 里写 `if x = 5:` 导致 `SyntaxError`。口诀:**一个"等于"装盒子,两个"等于"比谁大**。

### 知识点 1.2 `if` 语句:只有条件成立才执行

```python
age = 18
if age >= 18:                    # 冒号不能忘!
    print("你已经成年了")        # 缩进 4 格
    print("可以考取驾照")
print("这行始终执行")             # 没有缩进,不属于 if
```

> Python 用**缩进**(4 个空格)决定哪些语句属于哪个分支 —— 不像 C 语言用花括号。这是 Python 最标志性的语法。

### 知识点 1.3 `if-else`:二选一

```python
age = 16
if age >= 18:
    print("成年人")
else:                            # else 也始终带冒号
    print("未成年人")
```

### 知识点 1.4 `if-elif-else`:多选一

```python
score = 85
if score >= 90:
    print("A")
elif score >= 80:                # elif = else if,可写多个
    print("B")
elif score >= 70:
    print("C")
elif score >= 60:
    print("D")
else:
    print("F")
```

> 执行路径:`if` → `elif` → `else`,**从上到下第一个满足的分支执行**,执行完就跳出整个结构,后面的条件不再判断。

## 3. 第二讲(15 分钟) —— 逻辑运算与嵌套

### 知识点 3.1 逻辑运算 `and` `or` `not`:组合多个条件

| 运算符 | 含义 | 例子 |
|---|---|---|
| `and` | 两边都真才真 | `age >= 18 and age < 60` |
| `or`  | 一边为真就真 | `score < 0 or score > 100` |
| `not` | 取反 | `not is_raining` |

```python
age = 25
if age >= 18 and age <= 60:
    print("劳动年龄")

# 判断平年:不能被 4 整除,或能被 100 整除但不能被 400 整除
year = 1900
is_leap = (year % 4 == 0 and year % 100 != 0) or (year % 400 == 0)
print(is_leap)  # False
```

### 知识点 3.2 嵌套条件:条件里再挖条件

```python
age = 25
if age >= 18:
    if age >= 60:
        print("老年人")
    else:
        print("成年人")
else:
    print("未成年人")
```

> 🔴 教学红线(多层缩进可读性): 两层嵌套以内可用,超过两层建议用 `elif` + `and` 扁平化,否则缩进太多难读。

### 知识点 3.3 三元表达式:一行搞定简单二选一

```python
age = 18
status = "成年" if age >= 18 else "未成年"
print(status)
```

> 语法:`值1 if 条件 else 值2`。条件成立返回"值1",不成立返回"值2"。适合简单场景,**复杂逻辑不要强塞一行**。

## 5. 第三讲(15 分钟) —— `in` 运算符与综合实战

### 知识点 5.1 `in` 运算符:判断包含

```python
# 字符串包含判断
email = "zhangsan@163.com"
if "@" in email:
    print("看起来像邮箱")

# 成员判断(列表/元组/字典 key)
fruits = ["苹果", "香蕉"]
if "橘子" in fruits:
    print("有橘子")
else:
    print("没橘子")
```

### 知识点 5.2 综合实战 1:BMI 计算器

```python
h = float(input("请输入身高(米):"))
w = float(input("请输入体重(千克):"))
bmi = w / (h ** 2)
if bmi < 18.5:
    result = "偏瘦"
elif bmi < 24:            # 隐含 >= 18.5
    result = "正常"
elif bmi < 28:            # 隐含 >= 24
    result = "偏胖"
else:
    result = "肥胖"
print(f"BMI: {bmi:.2f},评级: {result}")
```

> 教学提示:这里的 `elif bmi < 24` 实际等于 `elif 18.5 <= bmi < 24`,因为前面的 `if bmi < 18.5` 已经拦下了所有偏瘦的游客。

### 知识点 5.3 综合实战 2:闰年判断

```python
year = int(input("请输入年份:"))
is_leap = (year % 4 == 0 and year % 100 != 0) or (year % 400 == 0)
if is_leap:
    print(f"{year} 是闰年")
else:
    print(f"{year} 不是闰年")
```

> 闰年口诀:**四年一闰,百年不闰,四百年又闰**。

## 易错点

1. **`=` 是赋值 `==` 是比较**,`if x = 5:` 必报错。
2. **缩进必须统一**,混用 Tab 和空格会報 `IndentationError`。
3. **`elif` 顺序敏感**,范围宽的(高分/高金额)条件要写前面。
4. **`or`/`and` 两边必须是完整表达式**,`x == 1 or 2` 是错误的,要写 `x == 1 or x == 2`。
5. **三元表达式只适合简单二选一**,复杂逻辑别硬塞一行。
