# Day03 · 条件分支

## 关键知识点
- 比较运算符：`>`、`<`、`==`、`!=`、`>=`、`<=`（`=` 赋值 vs `==` 比较）
- `if` 语句：冒号 + 缩进（4 格）
- `if-else`：二选一
- `if-elif-else`：多选一，从上到下第一个满足即跳出
- 逻辑运算：`and`、`or`、`not`
- 嵌套条件：条件里再挖条件（建议不超过两层）
- 三元表达式：`值1 if 条件 else 值2`
- `in` 运算符：判断字符串包含 / 成员判断
- 综合实战：BMI 计算器、闰年判断
