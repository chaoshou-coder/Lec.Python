# Day09 · 列表进阶

## 关键知识点
- 切片进阶:前 N 个 / 后 N 个 / 步长切片 `[::2]`
- 列表生成式:`[expr for x in iterable if cond]`
- 二维列表:嵌套循环遍历矩阵格式  - 成员判断 + 去重:`in` + `append`,保留首次出现
- 浅拷贝 / 深拷贝:`copy.copy()` / `copy.deepcopy()`
- 函数 + 二维列表 CRUD 综合(引入 `*args` 前奏)
