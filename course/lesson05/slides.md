## 1. 第一讲(15 分钟) —— `def` 定义与调用 + 四种形式

### 知识点 1.1 函数是什么:给一段代码起个名字

**函数 = 工具 + 说明书**。定义时写一次,调用时用无数次。

```python
def greet():                    # def + 函数名 + 括号 + 冒号
    print("Hello!")             # 函数体缩进
    print("欢迎学习 Python")

greet()                         # 调用:函数名 + 括号
greet()                         # 再调用一次
```

> 口诀:**定义时打包,调用时拆包**。没有调用,函数里面的代码不会执行。

### 知识点 1.2 函数四种形式

| 形式 | 参数 | 返回值 | 例子 |
|---|---|---|---|
| ① 无参无返回 | 无 | 无 | `print_hello()` |
| ② 有参无返回 | 有 | 无 | `print_name("小明")` |
| ③ 无参有返回 | 无 | 有 | `get_today()` |
| ④ 有参有返回 | 有 | 有 | `add(1, 2)` |

```python
def say_hi():                   # ① 无参无返回
    print("Hi!")

def greet(name):                # ② 有参无返回
    print(f"你好,{name}")

def get_pi():                   # ③ 无参有返回
    return 3.14159

def add(a, b):                  # ④ 有参有返回
    return a + b

# 调用
say_hi()                        # Hi!
greet("小明")                   # 你好,小明
pi = get_pi()
c = add(3, 5)
print(c)                        # 8
```

> 四种形式不要求一次全记住,但**必须会写 ④**(最高频),其他都会退化到 ④ 的组合。

## 3. 第二讲(15 分钟) —— 形参 / 实参 / 默认参数

### 知识点 3.1 形参 vs 实参

```python
def greet(name):        # name 是"形参"(形式参数),相当于占位符
    print(f"你好,{name}")

greet("小明")           # "小明" 是"实参"(实际参数),调用时传进去
greet("小红")           # 不同实参,同一段逻辑
```

> 形参是函数**定义时**写在括号里的"坑位",实参是函数**调用时**塞进去的"值"。

### 知识点 3.2 默认参数:有些参数"可选填"

```python
def greet(name="学员"):     # name 默认值为 "学员"
    print(f"你好,{name}")

greet()                      # 你好,学员(用默认值)
greet("小明")                # 你好,小明(覆盖默认值)
```

> 🔴 教学红线(默认参数必须放最后): 默认参数后面不能再跟非默认参数,否则 `SyntaxError`。例:`def greet(name="学员", age):` ❌,正确写法:`def greet(age, name="学员"):` ✅。

### 知识点 3.3 关键字参数:按名字传参

```python
def introduce(name, age, city):
    print(f"{name},{age} 岁,来自 {city}")

introduce(age=18, city="成都", name="小明")  # 顺序无关
```

## 5. 第三讲(15 分钟) —— `return`:提前退出与多值返回

### 知识点 5.1 `return`:带一个结果回家 + 立刻退出

```python
def check_age(age):
    if age < 0:
        return "年龄不能为负"    # 立即退出,后面语句不再执行
    if age < 18:
        return "未成年"
    return "成年"

result = check_age(20)          # "成年"
print(check_age(-1))            # "年龄不能为负"
```

> `return` 像"放学铃" —— 听到就停,后面代码不执行。一个函数可以有多个 `return`,但**只执行遇到的第一个**。

### 知识点 5.2 返回多个值:元组解包

```python
def calculate(a, b):
    return a + b, a - b, a * b  # 返回元组 (和, 差, 积)

sum_, diff, prod = calculate(10, 3)
print(sum_, diff, prod)         # 13 7 30
```

> Python 的"返回多个值"本质是**返回一个元组**,用逗号分隔的变量"解包"。

### 知识点 5.3 作用域:变量在函数里/外

```python
x = 10              # 全局变量:函数外都能用

def foo():
    y = 20          # 局部变量:只在函数内有效
    print(x)        # ✅ 能访问全局

foo()
# print(y)          # ❌ 报错:函数外访问局部

def bar():
    x = 99          # 这个 x 是局部变量,和全局的 x 没关系
    print(x)

bar()               # 99
print(x)            # 10 ← 全局 x 没被改
```

> 口诀:**函数里能读外面的,不能直接改外面**(除非加 `global`,本节不提倡,了解即可)。

## 易错点

1. **函数定义后不调用,代码不会执行** —— 写了 `def` 不等于"跑一次"。
2. **`return` 语句后面的代码不会执行** —— 像放学铃,听到就走。
3. **默认参数必须放在非默认参数后面** —— `SyntaxError`。
4. **函数内部不能直接修改全局变量** —— 要用 `global` 显式声明(本节了解)。
5. **`return a, b, c` 实际返回元组** —— 多值返回本质是单值(元组)。
