# Day05 · 函数入门

## 关键知识点
- `def` 定义函数 / 调用函数
- 函数四种形式：无参无返 / 有参无返 / 无参有返 / 有参有返
- 形参 vs 实参
- 默认参数（必须放最后）
- 关键字参数：按名字传参，顺序无关
- `return`：带结果回家 + 立刻退出（多 return 只执行第一个）
- 多值返回：元组解包（本质返回元组）
- 作用域：全局变量 vs 局部变量（函数里能读外面，不能直接改外面）
- 函数封装与模块化设计思想
