# Day06 · 列表与字典

## 关键知识点
- 列表创建：有序、可变、可重复
- 列表索引 + 切片（同字符串，支持逆序 `[::-1]`）
- 列表可变性（字符串不可变，列表可改）
- 列表 CRUD：`append()`、`insert()`、`extend()`、`pop()`、`remove()`、`clear()`
- 列表排序与查询：`sort()`（原地，返回 None）、`index()`、`count()`、`in`
- 字典创建：key-value 对（带 key 的快递柜）
- 字典取值：`d[key]` vs `d.get(key, 默认值)`
- 字典 CRUD：增 / 改 / 删（`pop`）
- 字典遍历：`keys()`、`values()`、`items()`（最强迭代方式）
