## 小 / 中型项目

**控制台购物车系统（`shopping_cart.py`）**
- 目标：商品列表 → 加入购物车 → 查看购物车 → 结算 → JSON 持久化
- 验收点：浏览商品、添加/删除、数量累加、结算总价、函数封装、异常加固、JSON 持久化（加分项）
- 建议时长：90 分钟

## 阶段复习要点

综合题覆盖：数据类型、算术运算、字符串处理、列表字典、循环、分支、嵌套、`f-string`、`functools.wraps`、文件 I/O、异常处理、OOP 基础、模块与高级特性。
