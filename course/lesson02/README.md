# Day02 · 数据类型与运算

## 关键知识点
- 算术运算符：`+`、`-`、`*`、`/`、`//`、`%`、`**`
- 比较运算符、逻辑运算符（`and`、`or`）
- 数据类型：`int`、`float`、`str`、`bool`
- 格式化输出（`%`、`format`、`f-string`）
- `random.randint()` 生成随机数
- `ord()`/`chr()` 与 ASCII 码转换
