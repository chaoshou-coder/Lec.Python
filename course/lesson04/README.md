# Day04 · 循环入门

## 关键知识点
- `while` 循环：起始值、终止条件、递增步进（死循环风险）
- `break` 与 `continue`：彻底跳出 vs 跳过本次
- `for item in 序列`：遍历字符串、列表
- `range(start, stop, step)`：生成整数数列（左闭右开）
- 累加器与累乘器：初始值（加 0 乘 1）
- `end=''` 参数控制输出不换行
- 嵌套循环：九九乘法表、图形打印
- 鸡兔同笼：循环 + 条件合作
- 菜单循环：`while True + break` 骨架
- 字符串重复：`*` 运算符
