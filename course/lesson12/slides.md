## 1. 第一讲(15 分钟) —— 矩阵乘法与线性代数

### 知识点 1.1 逐元素乘法 vs 矩阵乘法

```python
a = np.array([[1, 2],
              [3, 4]])
b = np.array([[5, 6],
              [7, 8]])

# 逐元素乘法(昨天学的)
print(a * b)
# [[ 5 12]
 # [21 32]]

# 矩阵乘法(线性代数,今天学)
print(np.dot(a, b))
# [[19 22]    ← 1*5+2*7=19, 1*6+2*8=22
 # [43 50]]   ← 3*5+4*7=43, 3*6+4*8=50

# @ 运算符(Python 3.5+,推荐写法)
print(a @ b)  # 和 np.dot(a, b) 等价
```

> 口诀:**`*` 是"对位乘",`@` 是"行乘列"**。

### 知识点 1.2 常用线性代数函数

```python
a = np.array([[1, 2],
              [3, 4]])

# 转置
print(a.T)
# [[1 3]
 # [2 4]]

# 行列式
print(np.linalg.det(a))  # -2.0

# 逆矩阵
print(np.linalg.inv(a))
# [[-2.   1. ]
 #  [1.5 -0.5]]

# 特征值与特征向量
eigenvalues, eigenvectors = np.linalg.eig(a)
print(eigenvalues)  # 特征值
```

> 🔴 教学红线(矩阵乘法顺序): 矩阵乘法**不满足交换律**,`A @ B` ≠ `B @ A`。初学者常把顺序写反,导致结果错误或形状不匹配。强调:"左行乘右列,顺序不能换"。

## 3. 第二讲(15 分钟) —— 统计函数与轴(axis)概念

### 知识点 3.1 聚合函数:对整个数组计算

```python
a = np.array([[1, 2, 3],
              [4, 5, 6]])

print(a.sum())    # 21  ← 所有元素求和
print(a.mean())   # 3.5 ← 所有元素平均
print(a.std())    # 1.707... ← 标准差
print(a.max())    # 6
print(a.min())    # 1
print(a.argmax()) # 5 ← 最大值的**扁平化索引**
```

### 知识点 3.2 axis 参数:沿哪个轴聚合

`axis=0` = 沿**行**方向(压扁行,结果长度 = 列数);`axis=1` = 沿**列**方向(压扁列,结果长度 = 行数)。

```python
a = np.array([[1, 2, 3],
              [4, 5, 6]])

print(a.sum(axis=0))  # [5 7 9]  ← 每列求和(把行压扁)
print(a.sum(axis=1))  # [ 6 15]  ← 每行求和(把列压扁)
```

> 🔴 教学红线(axis 方向混淆): 学员常把 `axis=0` 记成"对第 0 轴(行)操作"。正确理解:**axis=0 表示"沿第 0 轴方向操作",即跨行计算,结果是每列一个值**。记忆口诀:"axis=0 跨行 → 剩列"。

### 知识点 3.3 随机数生成

```python
# 设置随机种子(保证可复现)
np.random.seed(42)

# 均匀分布 [0, 1)
print(np.random.rand(3, 2))       # 3×2 随机矩阵

# 标准正态分布
print(np.random.randn(5))         # 5 个 N(0,1) 随机数

# 指定均值和标准差的正态分布
print(np.random.normal(10, 2, 5)) # 5 个 N(10,4) 随机数

# 均匀整数 [low, high)
print(np.random.randint(1, 7, 10))  # 10 个 1-6 的骰子点数
```

> 为什么需要 `seed(42)`: 机器学习实验需要**可复现**,设种子后每次运行得到相同随机数。

## 5. 第三讲(15 分钟) —— 文件读写

### 知识点 5.1 二进制格式:save / load

```python
a = np.array([[1, 2, 3], [4, 5, 6]])

# 保存为 .npy 格式(单数组)
np.save("data.npy", a)

# 加载
b = np.load("data.npy")
print(b)  # 完全恢复原数组

# 保存多个数组为 .npz 格式
np.savez("data.npz", x=a, y=a * 2)
data = np.load("data.npz")
print(data["x"])  # 原数组
print(data["y"])  # 翻倍数组
```

### 知识点 5.2 文本格式:loadtxt / savetxt

```python
# 从文本文件加载(类似 CSV)
data = np.loadtxt("data.csv", delimiter=",", skiprows=1)
# delimiter: 分隔符
# skiprows: 跳过前几行(跳过表头)

# 保存为文本
np.savetxt("output.csv", data, delimiter=",",
           header="col1,col2,col3", comments="")
# comments="" 默认 header 前加 "# ",设为空字符串去掉
```

> 🔴 教学红线(编码问题): `loadtxt` 默认用系统编码,中文 CSV 常报 `UnicodeDecodeError`。解决方案: 用 `encoding="utf-8"` 或 `encoding="gbk"` 参数。类比 Day13 文件读写时的编码坑 —— 统一 UTF-8 是最佳实践(references.md §3.3)。

## 易错点

1. **矩阵乘法不满足交换律**: `A @ B` ≠ `B @ A`,顺序不能随意调换。
2. **axis 方向**: `axis=0` 跨行计算(结果按列),`axis=1` 跨列计算(结果按行)。
3. **`argmax` 返回扁平化索引**: 多维数组 `argmax()` 返回"拉平后"的索引,要配合 `unravel_index` 还原。
4. **随机数可复现**: 机器学习实验必须 `np.random.seed(42)`,否则每次结果不同。
5. **文件编码**: 读取中文 CSV 时指定 `encoding="utf-8"` 或 `"gbk"`,避免 `UnicodeDecodeError`。
