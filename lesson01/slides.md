## 1. 第一讲(15 分钟) —— `print` 与 `Hello World`

### 知识点 1.1 搭建环境:Python + VSCode 跑通第一句

1. 管理员代装 python.org 3.10+ 和 VSCode(课前准备)。

```python

```

> 🔴 教学红线(中英文符号): 学员常混用中文括号`()`或中文引号`""`,导致 `SyntaxError`。要求切换到**英文输入法**再写代码。首次出现报错时,投影对比中英文符号差异。

### 知识点 1.2 `print()` 输出:一次打多个、换行与不换行

```python
print("刘备", "关羽", "张飞")  # 刘关张(空格分隔)
print("刘备" + "关羽")         # 刘备关羽(字符串拼接)
print("第一行\n第二行")          # \n = 换行
print("不换行", end="")         # end="" 取消默认换行
print("接续这一行")
```

> 口诀:**`print` 就是对话框,括号里写啥出啥**,多个值用逗号。

## 3. 第二讲(15 分钟) —— `input` 获取输入

### 知识点 3.1 `input()`:让电脑停下来等你打字

`input()` 让程序暂停,等你从键盘打字,按回车后把你打的**字符串**返回。

```python
name = input("请输入你的姓名:")  # 屏幕显示提示,等你输入
print("你好," + name)

age = input("请输入年龄:")       # ⚠️  age 现在是字符串 "18",不是数字 18
print(type(age))                 # <class 'str'>
```

> 记忆口诀:**`input` 永远是字符串**,需要数字请 `int()` / `float()` 裹一层。

## 5. 第三讲(15 分钟) —— 变量与命名

### 知识点 5.1 变量 = 给数据起个名字

```python
name = "小明"       # 把字符串 "小明" 装进 label 叫 name 的盒子
age = 18            # 整数
height = 1.75       # 浮点数
is_student = True   # 布尔
```

### 知识点 5.2 变量命名规则

- 由**字母、数字、下划线**组成,不能以数字开头
- 区分大小写 (`age` 和 `Age` 是两个变量)
- 不能是 `if` `for` `print` 等保留字
- **约定**:小写 + 下划线,如 `user_name`(snake_case)

```python
# 合法
user_name = "张三"
_age = 18
name2 = "李四"

# 非法(名副其实会报错)
# 2name = "张三"     # 数字开头
# my-name = "张三"   # 横杠不行
# if = 18            # 保留字
```

### 知识点 5.3 赋值与重新赋值

```python
x = 10
x = 20          # x 被重写为 20

a = 10
b = a           # b 复制了 a 的值
a = 99
print(b)        # 10  ← 改 a 不影响 b
```

> 🔴 教学误区(数学等号≠赋值号): 数学里 `x = x + 1` 永远不成立;Python 里`x = x + 1` 表示"让 x 在原来基础上加 1"。把 `=` 理解成"**把右边的结果装进左边的盒子**"。

## 易错点

1. **`input()` 永远是字符串**,要算数必须先 `int()` / `float()` 转换。
2. **中英文符号混用**,括号 `()`、引号 `""` 必须英文;中文 `（）` `""` 报错。
3. **变量名不能以数字开头**,`2name` 非法;`name2` 合法。
4. **`print("a" + 1)` 报错**,字符串和整数不能直接 `+`;要 `str(1)` 或 f-string。
5. **`=` 是赋值不是比较**,`==` 才是相等比较。
