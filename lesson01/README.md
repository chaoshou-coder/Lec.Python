# Day01 · Python 与开发环境

## 关键知识点

- 程序的概念、Python 解释器、VSCode / PyCharm
- `print()` 输出
- `input()` 接收用户输入
- 变量标识符 / 赋值 / 变量三要素(`type()` / `id()` / 值)
- 注释

## 当堂

- 01: 输出一段自我介绍
- 02: 输入姓名/年龄,打印"你是 X,今年 Y 岁"
- 03: 用 `type()` 观察 int / str / bool
- 04: 用 `id()` 观察不可变类型改变时地址的变化

## 课后

## 易错点

1. `input()` 返回字符串,算术运算前要 `int()`
2. 引号要成对
3. print 多个参数用逗号会自动加空格

## 教师备忘

- 先演示 → 再写 → 再模仿 → 纠错
- 不要讲"引用语义"概念,只讲"赋值"
