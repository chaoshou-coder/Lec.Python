## 3. 第二讲(20 分钟) —— 数据清洗三件套

### 知识点 3.1 去重:数据泄露是灾难

```python
# ❌ 错误:先划分再去重
训练集, 测试集 = 划分(全部数据)
训练集 = 去重(训练集)  # 同一条可能两边都有!
```

🔴 教学红线(数据泄露): **先去重,再划分** ——
否则模型分数虚高,上线后一塌糊涂。

```python
# ✅ 正确顺序
全部数据 = 去重(全部数据)
训练集, 测试集 = 划分(全部数据)
口诀: 先洗后切,先整后分
```

### 知识点 3.2:3.3 去噪 + 格式化

```python
def 清洗(数据列表):
    干净 = []
    for 条目 in 数据列表:
        # 去噪: 短、乱、缺、脏 四种
        if len(条目["review_text"]) < 5: continue
        if not 条目["title"]:           continue
        if "<" in 条目["review_text"]:  continue
        # 格式化: 统一类型
        条目["rating"] = int(条目["rating"])
        条目["price"] = float(
            条目["price"].replace("£", ""))
        条目["title"] = 条目["title"].strip()
        条目["sentiment"] = (
            "positive" if 条目["rating"] >= 4
            else "negative" if 条目["rating"] <= 2
            else "neutral")
        干净.append(条目)
    return 干净
```

🔴 教学红线(类型): 忘了转 int,
字符串比较 "10" < "2" 语义错乱!

> 噪音记忆:**短、乱、缺、脏**。

## 5. 第三讲(20 分钟) —— 质量评估

### 知识点 5.1:5.2 样本量 + 均衡性

```python
def 评估(数据列表):
    总数 = len(数据列表)
    print(f"总条数: {总数}(<100过少,<1千POC级)")
    统计 = {"positive": 0, "neutral": 0, "negative": 0}
    for 条目 in 数据列表:
        统计[条目["sentiment"]] += 1
    for 类别, 数量 in 统计.items():
        print(f"  {类别}: {数量} ({数量/总数*100:.1f}%)")
    if max(统计.values()) / 总数 > 0.8:
        print("⚠️ 严重不平衡!")
```

🔴 教学红线(偏差): 90% 正面 → 模型学会
"都猜正面就 90% 准"。数据有偏差就学歪。

> 口诀:**数据决定模型天花板** ——
>   数据科学家 80% 时间花在评估数据。

### 知识点 5.3 标注质量:标签不可盲信

```python
def 抽检(数据列表):
    import random
    样本 = random.sample(数据列表, min(10, len(数据列表)))
    for i, 条目 in enumerate(样本, 1):
        print(f"{i}. [{条目['sentiment']}] "
              f"《{条目['title'][:20]}》")
```

> 工业界用 Kappa 系数衡量标注一致性,
>   本节建立"标签不是天然可信"意识即可。

## 易错点

1. **数据泄露**: 先划分再去重。
   牢记"先洗后切"。
2. **类型不统一**: 评分价格保留字符串,
   比较必出错。int()/float() 必须做。
3. **忽略均衡性**: 90% 正面 → 模型猜正面
   就 90% 准。用前必须评估均衡性()。
4. **脏数据**: 噪音四象限 ——
   短、乱、缺、脏(filter 全上)。
5. **不可复现**: 没种子、没 requirements、
   没 README。标准不是"我能跑通"。
